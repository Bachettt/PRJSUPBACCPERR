3. Configurer l'Application Flask

Dans votre application Flask, configurez SQLAlchemy pour utiliser PostgreSQL :

from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'postgresql://user:password@db/mydatabase'
db = SQLAlchemy(app)


Construire et Lancer les Conteneurs

À la racine de votre projet, exécutez la commande suivante pour construire et lancer vos conteneurs Docker :

docker-compose up --build



Maintenant, vous devriez avoir une instance de votre application Flask tournant dans un conteneur Docker et connectée à une instance PostgreSQL dans un autre conteneur. Vous pouvez accéder à votre application Flask en ouvrant votre navigateur web et en naviguant vers http://localhost:5000.

N'oubliez pas de configurer les migrations SQLAlchemy et de les exécuter pour créer les tables nécessaires dans votre base de données PostgreSQL.
